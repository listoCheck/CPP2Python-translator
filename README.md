# C++ to Python Translator

## Описание проекта
Этот проект реализует транслятор, который переводит программы с языка C++ на Python. Транслятор читает исходный код из текстового файла (`input.txt`), анализирует его с помощью лексического и синтаксического анализаторов, строит абстрактное синтаксическое дерево (AST) и генерирует эквивалентный Python-код, который записывается в `output.txt`.  

Проект демонстрирует принципы компиляторов и интерпретаторов, включая:  

- **Лексический анализ** — разбиение исходного кода на токены.  
- **Синтаксический анализ** — построение AST для представления структуры программы.  
- **Генерация кода** — создание корректного Python-кода из AST.  
- **Поддержка комментариев** — C++-комментарии (`//`) конвертируются в Python-комментарии (`#`).
